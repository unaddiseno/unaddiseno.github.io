Tipos y ejemplos

Herramientas para pruebas dinámicas

Pruebas unitarias: Frameworks para pruebas unitarias como GUnit, Xunit

Integración: Generalmente son pruebas que se hacen manualmente.

Funcionales y regresión: herramientas como QTest, permiten a los Ingenieros de QA realizar y gestionar de forma fácil los casos de prueba. Frameworks para automatización de pruebas agilizan este proceso. Puede ser apoyada por software de gestión de bugs, como Mantis, Jira, etc.

Desempeño: Herramientas tipo profiler para código fuente, bases de datos etc. Existen aplicaciones que permiten probar el desempeño y capacidad de otra aplicación web para atender solicitudes de forma masiva.

Seguridad: Aunque existen herramientas automatizadas, generalmente se hace de forma manual, apoyándose en muchas herramientas diferentes para verificar la seguridad del sistema.

UAT: Se realizan de forma manual dada su naturaleza, puede ser apoyada por software de gestión de pruebas como QTest y de gestión de bugs, como Mantis, Jira, etc.
